6.1 分類問題とは¶

パーセプトロンは脳の神経細胞であるニューロンを模した分類モデルです｡非常に単純なモデルであるため､実際に利用されることはほとんどありません｡しかし､この後で学ぶロジスティック回帰やSVMあるいはニューラルネットワークなどの多くの有用なモデルの基本構造になっているという意味で学ぶ意義があります｡構造としては､線形分類器といってこれまで回帰問題を解くために利用していた線形モデルを､分類問題に適用しています｡基本的には2クラス分類を行いますが､それを応用して3クラス以上の多クラス分類にも適用できます｡

分析ストーリー : 植物の種分化研究¶

今回のストーリーはアカデミックな場面を想定したものです｡植物学の研究のひとつに､種分化研究という生物学的な種が誕生するまでの歴史的プロセスを解き明かそうとする領域があります｡とある植物学者は､種分化研究の題材にアヤメの花を選びました｡そのルーツを知るため､まずはアヤメの花の3つの品種について形状の違いを調査し､その結果を定量的に評価することを目指しました｡機械学習によって品種ごとの形状の基準がどこにあるのかを明確にします｡

irisデータ¶

irisデータとは､3品種×50個=計150個のアヤメ(iris)の花について｢がく片(sepal)｣と｢花びら（petal）｣のそれぞれについて長さと幅を記録したデータです｡これら4つの特徴量を説明変数として､品種を表すクラスラベルを予測します｡このデータは､米国の植物学者であるエドガー アンデルセンの研究によって収集され､のちにイギリスの統計学者であるロナルド･フィッシャーとの共同研究に利用されました｡

出典: Iris flower data set

分類問題とは､サンプルを複数あるクラスのどこに振り分けるかを考える問題です｡統計解析の領域では､判別分析とも呼ばれます｡回帰と違って､クラスラベルには連続性はありません｡

6.1.1 適用例¶

分類問題は､機械学習の手法の中でも特に非常に適用範囲が広いです｡あらゆる場面で利用され､事例にも枚挙に暇がありません｡

  • 送られてきたメールを､スパムメールかそうでないかを判別する
  • 病院に診察に来た患者が､癌にかかっているかを判定する
  • 紙に手書きで書いてある数字が､どの数字であるかを認識する
  • クレジットカードの取引の不正を検知する
  • インターネットサービスの顧客が解約をしそうかどうか判定する
  • ECサイトに出品された商品をカテゴリ分けして､自動でタグを付ける

6.1.2 分類問題の目標¶

回帰問題の目標が最適な回帰式を求めることだったのに対して､分類問題の目標は最適な決定境界を求めることです｡決定境界とは､クラスどうしの境目となる線のことです｡回帰直線を求めることで､説明変数がわかれば目的変数がわかるように､決定曲線を定めることで､説明変数がわかれば､クラスラベルがわかることを目指します｡結局は､回帰のときと同じように最適な直線ないし曲線の方程式を求めることができればいいので､やることは回帰のときとそれほど大きくは変わりません｡

6.2 コンセプトと定式化¶

この節では､パーセプトロンというものがどういったものなのか､そのコンセプトと定式化までの過程を紹介します｡

6.2.1 ニューロンの模倣¶

パーセプトロンは初期の人工知能研究の中で生まれた分類モデルであり､神経細胞であるニューロンを模した構造をしています｡ニューロンが電気信号によって情報の処理･伝達の機能を､数値計算に置き換えています｡

ニューロン¶

  1. 外部から電気信号が入力される
  2. 樹状突起でおこる電圧で入力が変化する
  3. 細胞体に電気信号が蓄積される
  4. ある一定量以上が蓄積されると発火し､他のニューロンに電気信号を送り出す

パーセプトロン¶

  1. 外部から説明変数が入力される
  2. エッジに格納されている重みパラメータにより入力が変化する
  3. ノードに線形入力 $\boldsymbol{w}^T \boldsymbol{x}$ が格納される
  4. 線形入力が一定量を超えない場合は-1を､超えた場合は+1を出力する

6.2.2 計算規則¶

パーセプトロンの計算は､線形モデルを用いて行われます｡回帰問題と同じように線形入力を計算して､あらかじめ設定したしきい値を超えるか超えないかを判定し､超えた場合は+1のラベルを､超えなかった場合は-1のラベルを出力します｡また､出力に変換を加える関数を活性化関数といい､今回利用している活性化関数はステップ関数（ヘビサイド関数）と呼ばれます｡

$$u = w_0x_0 + w_1x_1 + ... + w_dx_d$$

$$ \Phi(u) =\begin{eqnarray} \left\{ \begin{array}{l} +1\ \ (u \geq 0) ･･･class 0\\ -1\ \ (u < 0)･･･class 1 \end{array} \right. \end{eqnarray} $$

6.2.3 誤差関数の定義¶

パーセプトロンの学習規則は､これまでに扱ってきた二乗誤差関数での勾配降下法とは異なります｡予測と正解のクラスラベルで単純に差をとり､そこに学習率が掛け算されています｡また､形式上説明変数 $x_j$ も掛け算しています｡

$$\Delta \boldsymbol{w_j} = \eta (y^{(i)} -\Phi(u^{(i)})) x_j^{(i)}$$

パーセプトロンの場合､クラスラベルは±1で与えられていますので､ $y^{(i)} -\Phi(u^{(i)})$ の部分は以下の表のように定まります｡そのため､誤差は予測が当たっていれば0に､外れていれば±2になります｡これをもとに､誤分類の個数を減らそうとする修正が可能になります｡

6.2.4 線形分離可能な問題¶

パーセプトロンは､データが完全な線形分離が可能であるという前提を置いています｡パーセプトロンは間違いの個数に応じて修正を繰り返すので､線形分離ができない問題の場合は収束しません｡｢ああでもない､こうでもない｣と､行ったり来たりの学習を繰り返すことになり得ます｡こうした限界があることが､パーセプトロンの弱点です｡ちなみに､8章で紹介する分類モデルのSVMは､線形分離不可能な問題にも対応しています｡

定まった決定境界を可視化します｡そこで､2次元平面における決定境界を散布図上にプロットする関数を用意しました｡分類問題を解くときはそのまま利用してください｡5クラスまでの分類に対応するようになっています｡

# ２次元データの決定曲線をプロットする関数
# (X, y, 分類器)
def plot_decision_regions(X, y, classifier, resolution=0.02):
    from matplotlib.colors import ListedColormap
    markers = ("s", "x", "o", "^", "v")
    colors = ("red", "blue", "lightgreen", "gray", "cyan")
    cmap = ListedColormap(colors[:len(np.unique(y))])
    
    x1_min, x1_max = X[:, 0].min()-1 , X[:, 0].max() +1
    x2_min, x2_max = X[:, 1].min()-1 , X[:, 1].max() +1
    
    xx1, xx2 = np.meshgrid(np.arange(x1_min, x1_max, resolution), np.arange(x2_min, x2_max, resolution))
    Z = classifier.predict(np.array([xx1.ravel(), xx2.ravel()]).T)
    Z = Z.reshape(xx1.shape)
    plt.contourf(xx1, xx2, Z, alpha=0.4, cmap=cmap)
    
    plt.xlim(xx1.min(), xx1.max())
    plt.ylim(xx2.min(), xx2.max())
    
    for idx, cl in enumerate(np.unique(y)):
        plt.scatter(x=X[y == cl, 0], y=X[y == cl, 1], alpha=0.8, c=cmap(idx),marker=markers[idx], label=cl)

6.5 多クラス分類への拡張¶

2クラス分類を行う分類器を応用して､多クラス分類を実装します｡

6.5.1 分類の確信度合い¶

多クラス分類をするためには､分類問題における確信度合いというものを理解しておく必要があります｡とある2クラス分類において､確信度合いは境界線からの距離に､さらに言えばそれを表す線形入力の大きさに表れます｡境界線とは､分類において最も曖昧な位置です｡境界線に近いサンプルは､どちらのサンプルに属すかが際どいといえます｡逆に､境界線から離れれば離れるほど分類結果は明らかになるので､確信度合いが高くなるのです｡

6.5.2 一対他（One-vs.-Rest）¶

一対他とは､2クラス分類を行う分類器を応用して多クラス分類を実現するための手法です｡多数のクラスを｢ひとつのクラス｣と､｢その他のクラス｣の2つに分けて2クラス分類に持ち込みます｡これをすべてのクラスに対して行い､クラスと同じ数の2クラス分類器をつくります｡そして､最終的に線形入力の値が最も大きい分類器を採用します｡これは､図形的には各境界線の交点において角の二等分線を繋いで新たな境界線を作ることに相当します｡
