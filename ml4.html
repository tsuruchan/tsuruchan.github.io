4.1 多項式モデル¶

多項式回帰とは､その名の通り多項式（2次関数や3次関数など）を利用した回帰分析のことです｡これにより､回帰式を曲線（あるいは曲面､超曲面）にすることができます（｢非線形性を導入する｣といいます）｡

2､3章で扱った線形回帰のモデルでは､目的変数と説明変数との間に直線的な関係があることを前提にしていました｡しかし､変数間の関係が曲線的になっていることも､もちろんあります｡実際､前回の重回帰分析で扱ったBostonデータのLSTATは､目的変数であるMEDVと曲線的な関係にありました｡前回はこれを無理やり線形回帰に持ち込みましたが､当然､曲線で柔軟にフィットさせたほうが､より当てはまりの良いモデルになります｡今回は､前回のBostonデータ分析の延長で､非線形性を導入することによって予測性能が上がるのかを検証してみましょう｡

多項式とは､2次関数や3次関数のような高次の関数も含めた関数を指し､以下の式で一般化できます｡

$$y = w_0 + w_1x + w_2x^2 + … + w_dx^d$$

4.1.1 線形基底関数モデル¶

線形基底関数モデルとは､変数変換によって拡張された線形モデルです｡実は､多項式回帰も線形モデルにひと工夫加えるだけで実行できます｡下の式を見ていただければわかるかと思うのですが､ $x_1^2$ や $x_1^3$ を説明変数のひとつと見なせば､多項式モデルも線形モデルと何ら変わらない構造なのです｡ $x_1$ から $x_1^2$ や $x_1^3$ という変数をつくりさえすれば､あとは線形回帰と全く同じ方法で学習できます｡

4.1.2 基底関数¶

基底関数とは､線形モデルを拡張するための変数変換を行う関数です｡基底関数には代表的なものがいくつもありますが､今回は以下に表す多項式基底の基底関数を利用します｡やっていることは特に難しくなく､与えられた変数の累乗を計算して返しているだけです｡これを利用して､LSTATについて2次関数や3次関数で回帰できるようになります｡

$$\Phi(x_j) = x^j$$

4.2.4 自由度調整済み決定係数の比較¶

グラフによって､多項式回帰の方が当てはまりが良さそうだということがわかりました｡次に､それを定量的に確認するために自由度調整済み決定係数を比較してみます｡

In [48]:
# 自由度調整済み決定係数
# (決定係数, trainまたはtestのサンプル数, 利用した特徴量の数)
def adjusted(score, n_sample, n_features):
    adjusted_score = 1 - (1 - score) * ((n_sample - 1) / (n_sample - n_features - 1))
    return adjusted_score

4.3.5 残差プロットの比較¶

また､残差プロットを比較しても面白いことがわかります｡前回､残差の発生傾向のところで､プロットが山なりのカーブを描いていたことには触れませんでした｡実は､あのカーブは線形回帰がLSTATの曲線的な傾向を無視していたために生まれていたものだったのです｡多項式回帰の残差プロットでは､そのカーブが吸収されて､残差0の線の周囲により良くまとまっていることがわかります｡

In [57]:
# 残差プロットを行う関数を定義
def res_plot(y_train, y_train_pred, y_test, y_test_pred):
    # train, testごとに残差を算出
    res_train = y_train_pred - y_train
    res_test  = y_test_pred - y_test
    
    # train, testごとに(予測値, 残差)をプロット
    plt.figure(figsize=(8, 8))
    plt.scatter(y_train_pred, res_train, color='blue', marker='o', label='train', alpha=0.5)
    plt.scatter(y_test_pred, res_test, color='green', marker='s', label='test', alpha=0.5)
    
    plt.xlabel('Predicted Values')                  # x軸のラベルを表示
    plt.ylabel('Residuals')                         # y軸のラベルを表示
    plt.legend(loc='upper left')                    # 凡例を左上に表示
    plt.hlines(y=0, xmin=-10, xmax=50, color='red') # 残差0の線を表示
    plt.xlim([-10, 50])                             # 表示する横軸の範囲を指定
    plt.show()

4.4 過学習とのトレードオフ¶

動画リンク: 多項式回帰④¶

多項式回帰は､線形回帰よりも柔軟なフィッティングを可能にします｡しかし､その反面､柔軟さは過学習のしやすさにもなります｡極端な話､次数を大きくしていけば行くほどモデルの柔軟さが増し､trainデータへの当てはまりはどんどん良くなっていきます｡しかし､そのとき必ずしもtestデータへの当てはまりがいいとは限りません｡重回帰のときの繰り返しになりますが､教師あり学習で大事なのは将来のデータへの予測性能です｡なので､むやみに次数を上げることは､あまり好ましいことではありません｡必ず､testデータの自由度調整済み決定係数を基準に､ある程度の次数にとどめるようにしてください｡

次回予告¶

次回のテーマは正則化です｡正則化とは､モデルが複雑になりすぎることを抑える手法です｡過学習を抑制するためのテクニックとして知られており､回帰に限らずDeep Learningなどでも広く利用されるテクニックです｡先程指摘があったように､複雑なモデルは往々にして汎化性能が下がる原因になります｡次回は､今回学んだ多項式回帰を題材に､正則化について学んでいきましょう｡
